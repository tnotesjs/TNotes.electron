## 1. 📝 Summary
- demo - 自动截图程序
- 目前在测试 nodejs 的第三方库，看下是否能够支持实现定时定区域自动截图的功能，如果表现良好，可以考虑将其集成到 electron 中，出一个桌面端的自动截图工具。
- 应用场景：看外文视频的时候，自动定时截字幕。
- 功能：
  - 定时截图
  - 定区域截图
  - ORC 识别（识别字幕，去重）
